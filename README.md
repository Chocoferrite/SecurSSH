## What is SecurSSH?

SecurSSH is a cross-platform SSH client designed for developers, 
sysadmins, and DevOps teams. It centralizes SSH credentials, keys, 
and server addresses in a single AES-GCM end-to-end encrypted vault.

## Features

- 🔐 AES-GCM end-to-end encryption, zero-knowledge architecture
- 👥 Team permissions — Admin, Member, Viewer (RBAC)
- 🖥️ Multi-tab terminal with split view and local shell
- 📁 Built-in SFTP dual-pane file manager
- 🔗 Live collaborative sessions via web viewer
- 📋 Snippet manager and startup actions
- 🌍 Available in 10 languages
- 🇪🇺 EU-hosted, GDPR-native
- ✅ Free plan available
